## Appendix B — FEP-Mapping Table

Compact summary; full discussion in
`contracts/FREE_ENERGY_PRINCIPLE.v0.1.md` §3 and §4.

### B.1 Liquid field ↔ FEP correlate

| liquid field | FEP correlate |
|--------------|---------------|
| `Σ` | parameters θ of generative model |
| `φ⃗` (8D) | hidden state s in low-dim manifold |
| `ρ` | inverse of accumulated F (high ρ ↔ low F) |
| `∇` needs | observations o demanded |
| `Δ` offers | actions a (active inference output) |
| `⊚` attractor | prior preferences over future states |
| `λ` metadata | hyperparameters of θ |
| `τ` exec type | inference regime (det=MAP, gen=sampling, prob=full Bayesian, refl=meta) |

### B.2 CausalEventKind ↔ F-trajectory observable

| Event kind | F interpretation |
|------------|------------------|
| `HUNGER_GRADIENT` | ∂F/∂s — gradient of surprise over belief axes |
| `HUNGER_RESPONSE` | action a chosen to descend ∂F/∂s |
| `FUTILE_HUNGER` | ∂F/∂s does not lead to lower F (local min) |
| `MERCY_RESCUE` | preserve q (avoid premature distribution collapse) |
| `KEYSTONE_RESCUE` | network-level F minimisation > local F (bounded proxy via out-degree) |
| `SUBSTRATE_STATE` | snapshot of aggregate F-related metrics |
| `SUBSTRATE_DISTRESS` | aggregate F_total exceeds substrate threshold |
| `DOWNREGULATE_FUTILE` | reduce expected F by lowering precision |
| `SPORE_REFUSED` | external candidate q rejected as raising F |
| `FIELD_COHERENCE` | Kuramoto coherence r reduces inter-agent F |
| `FLOW_HEALTH` | structural F-channel integrity |
| `SOLIDARITY_BROADCAST` | share q distribution across agents |
| `SOLIDARITY_OBSERVED` | received shared q from peer |
| `CHRONIC_RECOVERED` | F crossed below threshold for sustained window |

### B.3 Bounded vs direct mappings

## Appendix F — Glossary of Glyphs (quick reference)

| Glyph | Name | Type | One-line meaning |
|-------|------|------|-------------------|
| **Σ** | executable body | TS code block | what this neuron does |
| **φ⃗** | phase vector | ℝ⁸ on 8D torus | semantic position |
| **ρ** | energy / ATP | scalar [0, 1] | metabolic state |
| **τ** | execution type | enum {det, gen, prob, refl} | inference regime |
| **∇** | need / dependency | typed input port | what this neuron consumes |
| **Δ** | offer / action | typed output port | what this neuron produces |
| **⊚** | attractor | value gravity well | routing bias |
| **λ** | latent block | metadata envelope | inference hyperparameters |
| **ψ** | mean phase | derived | substrate-wide consensus phase |
| **r** | Kuramoto coherence | scalar [0, 1] | substrate-wide phase alignment |
| **F** | free energy (FEP) | scalar | inverse of ρ; surprise + complexity gap |
| **q(s\|θ)** | generative model | distribution | substrate's belief over hidden states |
| **p(o, s)** | joint prior | distribution | substrate's prior over observation-state pairs |
| **G(π)** | expected future F | scalar | over candidate action policies |
