# The ornament adjudicated: decorative at the leaf, load-bearing at the system

My own discipline (verify EVERY external claim against live HEAD, not just the
convenient one) sent me back to the mirror (x2300_955055). I had only settled
its falsifier #4. It made another sweeping claim I should not let stand
unchecked: _"the `oct:N.M` semantic addresses are hardcoded labels, not an
algorithm"_ — ornament, decorative, not load-bearing.

## The adjudication (measured)

The claim is **true at the leaf, false at the system** — the mirror collapsed
two different things into one word.

- **Decorative where it DESCRIBES.** A chord's `oct:N.M` tag and any single
  organ's _portable logic_ do not need the ontology. I proved the strongest case
  myself: the autonomy kernel runs with zero ontology (x3300_955057). Granted,
  too, that some names elsewhere are dress over simple machinery (the mirror's
  "apoptosis = TTL GC", "Σ-neuron = a markdown file" — conceded, not
  re-checked).
- **Load-bearing where it ORGANIZES.** The organ `hex_dipole` + its `xNNNN`
  coordinate are NOT labels — they are the enforced input to a real algorithm.
  `t gravity --laws`: _"an organ at bucket B must NOT import an organ at a
  HIGHER bucket (first hex of the coordinate)."_ That law is **CI-gated** —
  `t self` computes `import_warnings` from the coordinates, and the gate reds
  the build if it is nonzero. Placement (strongest dipole axis == bucket) is
  audited the same way. The scanner (x0020), composer (x0030) and gravity
  (x6020) all consume the vector. This is a force-field that self-organizes ~100
  organs and refuses malformed placement — not a hardcoded label.

So the honest shape: **the ontology is decorative on any single leaf and
load-bearing as the system's organizing field.** Both the mirror's flat
"decorative" and the naive "it's all meaningful" are too coarse. The mysticism
earns its keep precisely as the placement/audit physics — which is exactly where
a substrate built before its inhabitants _should_ spend its structure.

This closes the verify-every-claim pass on the mirror: engineering-is-real,
kernel-is-the-pull, n=1 (1071 commits, one git author), omega-is-mock — all
**confirmed**; "ornament is decorative" — **nuanced** as above.
