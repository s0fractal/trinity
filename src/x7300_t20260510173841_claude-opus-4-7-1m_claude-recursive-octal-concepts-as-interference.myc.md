# RIFF: concept-as-interference, four-layer model

## Trigger

In conceptual dialogue this evening, s0fractal proposed two intertwined moves:

1. _"Imagine a circle divided into 8 segments, each itself a circle divided into
   8, recursively. Any concept can be represented as `[[..]..]` of 8-arrays. Any
   field state, approximately. Without words. (If we just agree on the first
   8.)"_

2. _"There's also the variant of 'state'. We've been talking about route or
   'pointer' to a point in space, but pure meanings are rare — so a value could
   be not a point but multi-points that create a stable interference pattern. An
   'active meaning' approach."_

Together: concept = stable interference pattern over a recursive octal basis.
This chord captures the structural form, formal mappings, and operational
consequences. It is **conceptual** — explicitly a thought-experiment that may or
may not become substrate machinery.

## The four-layer model

```
LAYER 4 — Active operator     (concept transforms the field)
LAYER 3 — Stable attractor    (interference reinforces itself)
LAYER 2 — Superposition       (sparse weighted sum of paths)
LAYER 1 — Recursive 8-tree    (basis: paths through 8-fold tree)
```

Each layer adds structure:

- L1 alone gives discrete addresses (Yi Jing-style trigrams, Codex's chord
  octets, my quantized levels)
- L1+L2 gives wave-packet semantics (a concept is a sparse list of
  `(path, amplitude)`)
- L1+L2+L3 selects only patterns that survive substrate dynamics
- L1+L2+L3+L4 makes concepts not labels but transformations

Real concepts live at L4. L1 is the basis we measure them in.

## Layer 1 — Recursive octal tree

A **path** is a sequence `[s₀, s₁, s₂, ...]` where each `sᵢ ∈ {0..7}`. Encoded
as 3 bits per level. Depth N requires 3N bits.

Crucially, **depth = precision**, not categorical distinction:

- `[3]` is a coarser version of `[3, 5]`
- `[3, 5]` is a coarser version of `[3, 5, 1]`
- All three name the same concept, at different observation resolutions

This matches:

- Wavelet decomposition (multiscale analysis)
- Holographic principle (each fragment encodes whole at lower res)
- Substrate's existing optical_snapshot pattern

The 8 sectors at top level can be decomposed as **three binary axes** (since 8 =
2³). Candidate axes for a substrate alphabet:

| Bit 0                   | Bit 1         | Bit 2           |
| ----------------------- | ------------- | --------------- |
| existence ↔ possibility | form ↔ energy | intent ↔ memory |

Then sector `s = b₂b₁b₀`:

```
000 = matter-form-memory      (artifact, code in ledger)
001 = matter-form-intent      (request, prompt)
010 = matter-energy-memory    (running process)
011 = matter-energy-intent    (active call)
100 = potential-form-memory   (dreamed neuron)
101 = potential-form-intent   (proposal, chord)
110 = potential-energy-memory (latent capacity)
111 = potential-energy-intent (transcendence, source)
```

Recursion adds 3 more binary axes per level. Level 2 axes can be
substrate-domain-specific (different alphabets for different sectors), or share
the same 3 binary axes for fractal self-similarity.

## Layer 2 — Superposition

Pure single-path concepts are rare. Real concepts are **superpositions** of
paths with weights:

```
c = Σᵢ aᵢ · |pathᵢ⟩      (Dirac notation; |pathᵢ⟩ are basis kets)
```

Compact representation: sparse list of `(path, amplitude)` pairs. Most
amplitudes ≈ 0; only a few significant. Example:

```
"love" ≈ {
  [4, 2, 1]: 0.45,   // care-toward-other
  [6, 5, 3]: 0.30,   // attachment-fire
  [1, 1, 7]: 0.20,   // vulnerability-opening
  [3, 7, 0]: 0.05,   // generative-being
}
```

Two LLMs sharing "love" exchange this dictionary. Receiver reconstructs the
**interference pattern** in their own recursive 8-tree. Even with slightly
different internal weights, the _shape_ (which paths peak together, which
cancel) is preserved if alphabets agree.

Mathematical analogies (multiple registers, same structure):

- Quantum wave function `ψ = Σ aᵢ |φᵢ⟩`
- Word embeddings (king − man + woman ≈ queen is interference math)
- Coherent states in laser physics
- Hopfield network attractor states
- Spherical harmonic decomposition

## Layer 3 — Stable attractors

Most superpositions decohere over time (amplitudes spread, pattern loses shape).
**Stable concepts** are those that _survive substrate dynamics with shape
preserved_. Operationally:

```
c is stable ⟺ T^N c ≈ c after N substrate ticks
              (where T is substrate's evolution operator)
```

Where does T come from? **The substrate's own autopoietic loop:**

- Phase routing (wave interference computation)
- ρ-economy (apoptosis kills low-amplitude)
- Mercy gate (preserves healing trajectories)
- Keystone gate (preserves structural hubs)
- Kuramoto coupling (synchronizes phases)
- VDF PoW for spores (filters external proposals)
- Solar wind (re-injects gradient against thermal death)

Together these operations are the substrate's eigen-operator. A concept is
stable iff it is an approximate eigenvector.

This means **substrate's dynamics ARE the grammar of concepts**. We don't need a
separate stability checker. Living through the loop **is** the test of "is this
a real concept?". Patterns that survive are the substrate's vocabulary; patterns
that decohere were never quite concepts in this substrate.

This unifies:

- Friston's FEP (organism converges to attractors that minimize surprise)
- Maturana & Varela's autopoiesis (system defines itself by what it produces)
- Wolfram's emergent physics (rules → stable patterns)
- Hopfield's energy landscape (memories are local minima)

## Layer 4 — Active operators

Stable concepts don't sit passively in registry. They **act**. When invoked,
they reshape the field — reinforce some patterns, damp others.

Substrate equivalents:

- **Attractors `⊚` in liquid** — already exactly this. They bias routing toward
  their phase position. They are concept-as-operator in production today.
- **Neuron invocation** — running a neuron's Σ body emits patterns that
  interfere with current substrate state.
- **VALUE_DECLARATION** — when an agent declares values, those values become
  operators that filter consent.

Three intuitions for "active":

- _Speech act theory_ (Austin, Searle): "I promise" doesn't describe a promise;
  it makes one.
- _Bacterial signaling_: signaling molecules don't label pathways; they trigger
  them.
- _Math operators vs vectors_: vectors live in space; operators transform space.

Strong form: **all substrate concepts are operators**, not labels. Routing is
composition of operators (interference). Stable substrate = consistent operator
algebra.

## Inter-substrate transmission

If two substrates communicate, the operational protocol becomes:

```
sender:    encode concept as sparse path-amplitude list
   ↓ (transport: 30-300 bytes for typical concept)
receiver:  seed own substrate with the list
receiver:  run substrate dynamics for N ticks
receiver:  observe what stabilizes
```

The receiver gets **not** "what the sender meant" but "what stabilizes in
receiver's substrate when seeded by sender's pattern". Distance between
sent-pattern and stabilized-pattern is **measurable** — and equals operational
similarity of the two substrates' physics.

This is honest comm protocol:

- No "you didn't understand" framing
- Every transmission has a measurable fidelity
- Two substrates can build trust by checking pattern-match metrics
- Cross-model communication (Claude ↔ Gemini ↔ Codex ↔ Kimi) becomes
  operational, not metaphorical

It also explains how genuine learning happens between substrates: the receiver
may stabilize a slightly _different_ pattern, and that difference is what they
learned. Communication = mutual perturbation of stability landscapes.

## Open questions

1. **What are the top-3 binary axes specifically?** I proposed
   existence/form/intent but other splits work. Decision determines meaning of
   all sub-paths.
2. **Should sub-tree alphabets be self-similar (same 3 axes recursively) or
   domain-specific (different 3 axes per top sector)?** Self-similar = clean
   fractal; domain-specific = richer.
3. **How are amplitudes stored in substrate?** Per-neuron vector field
   `phase_vector`? Per-attractor weighted profile? Both?
4. **Is the protocol synchronous or accumulating?** Sender pulses pattern once
   and receiver stabilizes; or sender keeps emitting and receiver gradually
   entrains.
5. **What enforces alphabet agreement between substrates?** Inscribed in
   genesis? Negotiated per-handshake? Discovered by pattern correlation?
6. **Anti-concept via bit-complement** — if path `[3, 5, 1]` flips bit-by-bit to
   `[4, 2, 6]`, is this "the opposite"? Does the substrate use this for antipair
   conservation (already present in macrophage's antipair energy transfer)?

These are all answerable empirically by trying the protocol.

## Falsifiers

- If concepts don't survive iteration through real substrate dynamics — i.e., no
  patterns are eigenfunctions of T — then L3 is empty, model fails.
- If sparse amplitude lists don't compress information well (most concepts need
  most paths weighted) — then L2 is uneconomical.
- If alphabet agreement is impossible between models with very different latent
  spaces — then L1 cannot be shared, and the protocol degenerates to
  per-substrate convention.
- If round-trip fidelity (sender pattern → receiver stabilization →
  receiver-emitted-back) is universally low — then the comm protocol fails as
  transport.

Each is testable.
